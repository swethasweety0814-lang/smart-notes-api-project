# Smart AI Note Manager 📝

A high-performance backend system built with **FastAPI** and **Python**, featuring a modern **Glassmorphism UI**.

## 🚀 Features
* **Full CRUD:** Create, Read, Update, and Delete notes seamlessly.
* **AI Integration Ready:** Structured for Gemini API integration for note summarization.
* **Modern UI:** Beautiful, responsive frontend using CSS Glassmorphism.
* **Clean Architecture:** Separated concerns with FastAPI, SQLAlchemy, and modular utility folders.

## 🛠️ Tech Stack
* **Backend:** FastAPI, Python 3.x
* **Database:** SQLite / SQLAlchemy ORM
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript
* **Server:** Uvicorn
